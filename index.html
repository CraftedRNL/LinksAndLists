Ordered Lists

The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.
  1. Ounce
  2. Pound
  3. ton

Unordered Lists

The ul tag defines an unordered (bulleted) list. Use the ul tag together with the li tag to create unordered lists.
  • Cow
  • Pig
  • Chicken

Definition Lists

The dl tag defines a description list. The dl tag is used in conjunction with dt (defines terms/names) and dd (describes each term/name)
Pencil
-Something you hold to write
Pencil Sharpener
-Something you use to sharpen a Pencil

Nested List

A list inside of a list. A list-ception. You would use this type of list b/c it lets you use it as the foundation for navigation menus since they determine the website's hierarchical structure

Comparing and contrasting

Ol tag is ordered and lets you put everything into steps, like cooking you needs steps.
Ul tag on the other hand doesn't do that and is just used for like a shoping list that doesn't usually require you to but in steps.
Dl tag however is differet from the other two as it has two tags inside it, one like a regular li tag and the other defined that one thing
However they are all similar as they all are some kind of list, although different in their use, still doing list stuff.
